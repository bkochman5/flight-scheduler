# Flight Scheduler

Full-stack flight scheduling system built as a learning project/

# Goals
- Practice backend development
- Practice frontend development
- Understand system design and business logic

# Planned Features
- Create and manage flights
- Seat allocation by class (first, business, economy)
- Passenger booking and cancellation
- Waitlists per seat class
- Passenger and flight status views

# Tech Stack (planned)
- Backend: PHP (Laravel)
- Frontend: React + TypeScript
- API: REST
